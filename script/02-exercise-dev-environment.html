Visual Studio Code Installieren

  1. Download: Gehen Sie zur offiziellen Visual Studio Code Website (https://code.visualstudio.com/) und laden Sie die für Ihr Betriebssystem passende Version herunter.
  2. Installation: Öffnen Sie die heruntergeladene Datei und folgen Sie den Installationsanweisungen. Standardmäßig schlägt die Installation vor, VS Code in Ihrem Anwendungsverzeichnis zu speichern.
  3. Erste Schritte: Nach der Installation öffnen Sie VS Code. Sie werden mit einem Willkommensbildschirm begrüßt, der Links zu Dokumentationen, Einstellungen und der Möglichkeit, neue Dateien zu öffnen oder Projekte zu starten, enthält.

Neues Repository auf GitHub erstellen

Github sollte Ihnen bekannt sein. Erstellen Sie ein neues öffentliches Repository für diesen Kurs. Die Abgabe Ihrer Portfolioaufgabe wird über dieses Repository erfolgen!

VS Code mit GitHub verbinden und Repository auschecken

  1. Git Installieren: Stellen Sie sicher, dass Git auf Ihrem Computer installiert ist. Falls nicht, laden Sie es von https://git-scm.com/ herunter und installieren Sie es.
  2. VS Code öffnen: Öffnen Sie VS Code und navigieren Sie zum "Reiter" Source Control. (Menü auf der linken Seite)
  3. Klonen Sie das erstellte Github Repository

Eine kleine Testseite erstellen

  1. HTML-Datei Erstellen: Im VS Code, erstellen Sie eine neue Datei namens index.html.
  2. Hello World Seite: Fügen Sie HTML-Code ein, sodass die Seite Hello World darstellt. Diese grundlegende HTML-Syntax sollte Ihnen bereits bekannt sein.

Testseite mit GitHub Pages veröffentlichen

  1. Commiten und Pushen Sie die Änderungen über den "Reiter" Source Control.
  2. GitHub Pages Aktivieren: Gehen Sie auf die GitHub-Repository-Seite, navigieren Sie zu „Settings“ -> „Pages“, wählen Sie den „master branch“ und speichern Sie.
  3. Haben Sie etwas Geduld und laden Sie die Seite neu. Sie sollten nun auf der Seite die URL zur deployten Webseite sehen.
  4. Tragen Sie den URL zur Seite im Kursnotizbuch ein!
