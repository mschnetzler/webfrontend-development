HTML Grundlagen

Einleitung

HTML ist die Grundlage für das Erstellen von Webseiten und definiert deren Struktur und Inhalt. Der Standard für HTML wird vom World Wide Web Consortium (W3C) festgelegt und kontinuierlich weiterentwickelt, um unterschiedlichen Browsern und Benutzerbedürfnissen gerecht zu werden. Weitere Informationen zur Definition von HTML finden Sie unter W3C Semantics.

XML

XML ist eine flexible Datenformatierungssprache, die für die Darstellung verschiedener Datenstrukturen verwendet wird. Im Kontext von Webseiten ermöglicht es die Beschreibung und den Transport von Informationen in einer Weise, die sowohl für Menschen als auch für Maschinen lesbar ist. Hier sind die Schlüsselkomponenten des XML-Grundgerüsts:

  • Elemente: Elemente sind die Bausteine von XML-Dokumenten. Ein Element wird durch ein Start-Tag wie <name> eingeleitet, kann Inhalte enthalten und wird durch ein End-Tag wie </name> abgeschlossen. Wenn ein Element keine Inhalte hat, kann es auch in verkürzter Form geschrieben werden: <name/>.
  • Tags: Tags markieren den Anfang und das Ende eines Elements. Sie bestehen aus spitzen Klammern mit einem Namen innen. Der Name innerhalb der Start- und End-Tags muss exakt übereinstimmen.
  • Attribute: Attribute liefern zusätzliche Informationen über Elemente. Sie befinden sich immer im Start-Tag eines Elements und haben die Form attributename="wert". Zum Beispiel: <note importance="high">. Hier ist "importance" der Attributname und "high" der Wert des Attributs.
  • Wurzelelement: Jedes XML-Dokument enthält ein einzelnes Wurzelelement, das alle anderen Elemente umfasst. Dieses Wurzelelement ist das oberste Element in der Hierarchie.

Das Grundgerüst

Eine HTML-Seite besteht aus mehreren grundlegenden Elementen, die ihre Struktur definieren:

  • <!DOCTYPE html> kennzeichnet die Dokumenttyp-Deklaration und ist notwendig, um dem Browser mitzuteilen, dass es sich um ein HTML5-Dokument handelt.
  • Das <html> Tag ist das Wurzelelement und umschließt den gesamten Inhalt der Webseite.
  • Innerhalb des <html> Tags finden wir das <head> Element, das für Metadaten wie den Titel der Seite, Verweise auf Stylesheets und andere Informationen genutzt wird, die nicht direkt auf der Seite angezeigt werden.
  • Das <body> Element beinhaltet den sichtbaren Inhalt der Webseite, wie Texte, Bilder, Links und mehr.

Beispiel:

<note importance="high" date="2024-03-31">
    <to>Benutzer</to>
    <from>Automatisierter Schreiber</from>
    <heading>Erinnerung</heading>
    <body>Bitte vergessen Sie nicht, die Blogpost-Anleitung zu lesen.</body>
</note>

Grundelemente

In diesem Abschnitt werden die Grundelemente einer HTML-Seite vorgestellt und kurz beschrieben:

Element Beschreibung
<div> Ein Block-Element, das zum Gruppieren von Inhalten und Strukturierung der HTML-Seite verwendet wird.
<span> Ein Inline-Element zur Formatierung eines kleinen Teils von Text oder zur Gruppierung von Elementen für stilistische Zwecke.
<br /> Ein Selbstschließendes Tag zur Erzeugung eines Zeilenumbruchs.
<p> Definiert einen Absatz und wird für den Haupttext auf einer Webseite verwendet.
<h1> bis <h6> Überschriftentags zur Darstellung von Titeln und Unterüberschriften, wobei <h1> die höchste und <h6> die niedrigste Ebene ist.
<ul>, <ol>, <li> Listenelemente für ungeordnete (ul) oder geordnete (ol) Listen. Jeder Listeneintrag wird mit <li> markiert.
<a href=""> Definiert einen Hyperlink, der zu einem anderen Dokument oder einer anderen Ressource führt.
<img src="" alt=""> Bindet ein Bild in das Dokument ein, wobei das 'src'-Attribut den Pfad zur Bilddatei und das 'alt'-Attribut einen Alternativtext enthält.
