Cascading Stylesheets (CSS)

Ohne CSS wären Webseiten nur einfache Textdokumente. CSS gibt uns die Möglichkeit, Webseiten ansprechend und interaktiv zu gestalten, was für eine positive Benutzererfahrung unerlässlich ist.

Wie funktioniert CSS?

CSS kann auf drei Arten in eine Webseite eingebunden werden:

  • Inline-Stil: Direkt in den HTML-Elementen über das "style" Attribut.
  • Interne Stylesheets: Im <head> Bereich der HTML-Datei mithilfe des <style> Tags.
  • Externe Stylesheets: Durch Verlinkung einer externen CSS-Datei im <head> Bereich mit dem <link> Element.

Grundlegende Syntax

Ein CSS-Regelwerk besteht aus einem Selektor und einem Deklarationsblock:

selektor {
    eigenschaft: wert;
}

Der Selektor gibt an, auf welches Element oder welche Elemente sich die Regeln beziehen. Der Deklarationsblock, umschlossen von geschweiften Klammern, enthält eine oder mehrere Deklarationen, die durch Semikolon getrennt sind. Jede Deklaration besteht aus einer Eigenschaft und einem Wert.

Ein einfaches Beispiel

Um die Textfarbe eines <p>-Elements in Blau zu ändern, würdet ihr folgenden CSS-Code verwenden:

p {
    color: blue;
}

Informationen zu CSS

Wenn Sie Informationen zu einzelnen Eigenschaften benötigen besuchen Sie die Webseite: https://www.w3schools.com/css/. Beachten Sie auch die kompatibilitäten einzelnere Browser für die Eigenschaften.

CSS anwenden

Um CSS in eure Webseite einzubinden, empfehle ich die Nutzung eines externen Stylesheets, da dies die Wartung erleichtert. Erstellt eine Datei mit der Endung .css (z.B. styles.css) und verlinkt diese in eurem HTML-Dokument im <head>-Bereich:

<link rel="stylesheet" href="styles.css">

Größeneinheiten in CSS

In CSS gibt es verschiedene Einheiten, um Größen wie Breite, Höhe, Schriftgröße und mehr zu definieren:

  • Absolute Größeneinheiten wie 'px' (Pixel), 'pt' (Punkte), 'cm' (Zentimeter), bleiben immer gleich.
  • Relative Größeneinheiten wie 'em' und 'rem' beziehen sich auf Schriftgrößen, während '%' sich auf ein Verhältnis zum umgebenden Element bezieht.
  • Viewport-bezogene Einheiten wie 'vw' (Viewport-Breite) und 'vh' (Viewport-Höhe) sind prozentuale Einheiten, die sich auf die Größe des Browserfensters beziehen.

Die Wahl der Einheiten hängt vom Kontext ab und davon, wie flexibel die Webseite auf verschiedene Bildschirmgrößen reagieren soll.
